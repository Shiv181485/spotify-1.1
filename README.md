## 📖 About

**Pulse** is an interactive Spotify analytics dashboard built with **Python** and **Streamlit**. It transforms music data into meaningful visual insights, allowing users to explore artist performance, track popularity, genres, and audio characteristics through an intuitive interface.

---

## ✨ Key Features

* 🎵 Spotify Music Analytics Dashboard
* 📈 Interactive Charts & Visualizations
* 🎤 Artist & Album Insights
* 💿 Track Popularity Analysis
* 🎼 Genre Distribution
* 🎧 Audio Feature Exploration
* 📊 Data-Driven Reports
* ⚡ Fast, Responsive & Interactive UI

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Shiv181485/spotify-1.1.git
```

### Navigate to Project

```bash
cd spotify-1.1
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```
