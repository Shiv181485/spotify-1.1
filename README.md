# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/Shiv181485/spotify-1.1.git
```

Move into the project

```bash
cd spotify-1.1
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
